Instantiates a RandomTimer object.

Arguments
model (despy.core.model.Model)
The RandomTimer must be assigned to a Model object.
name (String)
A short descriptive name for the RandomTimer object.
distribution A frozen scipy.stats discrete distribution.
Random intervals will be generated from this distribution.
callback Python function or method object
Callback function will be executed at intervals generated by RandomTimer object.
immediate Boolean
Optional. If True, first RandomTimerEvent will occur immediately. If False (default), first event will occur after a random interval.
priority Integer
Optional. Default is Priority.STANDARD. RandomTimerEvents will execute with this priority.
description String
Optional. Default is None. A paragraph that describes the RandomTimer object.
