HtmlReport.divisions¶

Python list containing simulation output. Read-only.

Each section of simulation output will be rendered as an HTML <div> element. The list can be appended to or have items deleted from it. Because it is read-only, it cannot be replaced with another list.

Type: Python List of Datatype formatted tuples.
