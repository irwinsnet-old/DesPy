Type: Boolean Default Value: True

Generator.folder_basename¶

Folder where output reports and graphs will be placed.

If None (the default value), the simulation will not generate any output or trace files. The value stored in folder_basename should be an absolute reference. For example:

gen.folder_basename = "C:/despy_output/resource_sim"

The Generator object will add a time-stamp to the end of the folder name when generating the output files. This allows the use to run the simulation several times without overwriting data from previous simulations.
